# ECS Tower Defense

Tower defense в изометрической перспективе: игрок размещает башни на поле и отражает волны врагов. Проект на ранней стадии — сейчас реализован MVP с размещением объектов через радиальное меню.

## Жанр

- **Tower Defense** — стратегия с фиксированным полем боя
- Размещение башен и других объектов на карте
- Изометрическая камера с управлением обзором

## Технологии

| Область | Стек |
|--------|------|
| Движок | Unity 6 (6000.3) |
| Рендер | Universal Render Pipeline (URP) |
| Архитектура | Собственный ECS-слой (`EntityWorld`, компоненты) |
| UI | UI Toolkit (радиальное меню) |
| Ввод | Unity Input System |
| IDE / AI | Cursor IDE, MCP for Unity |
